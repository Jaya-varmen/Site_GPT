# GPT-5.2 Visual Chat

Minimal ChatGPT-style UI with screenshot upload, powered by the OpenAI Responses API.

## Setup

1. Copy `.env.local.example` to `.env.local` and set `OPENAI_API_KEY`.
2. Install dependencies: `npm install`.
3. Run locally: `npm run dev`.

## Production

- Build: `npm run build`
- Start: `npm run start`

This app is ready for deployment on any Node.js hosting platform that supports Next.js.
